### 12.3 代码块中的格式

```python
# 这是一个包含格式的注释
def process_data(data):
    """
    处理数据的函数

    Args:
        data: 要处理的**数据**

    Returns:
        处理后的数据
    """
    # 检查数据有效性
    if not data:
        raise ValueError("数据不能为空")

    # 处理逻辑
    result = []
    for item in data:
        if item.get('active'):  # 只处理激活的项目
            processed = {
                'id': item['id'],
                'name': item['name'].upper(),  # 转换为大写
                'status': 'processed'
            }
            result.append(processed)

    return result
```
